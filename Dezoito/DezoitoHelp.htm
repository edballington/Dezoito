Rules for Playing Dezoito:

Rule 1: Dezoito boards contain overlapping squares (outlined by solid-lines surrounding a small, dark square) with each square subdivided into eight triangular cells. The eight cells within each square must contain numbers 1-8 without duplicates.  If the Solution Guide setting is activated, the small center square will turn green when a square contains values 1-8 without duplicates. This does not guarantee number placements are correct to solve the entire puzzle, only that Rule 1 is satisfied.

Rule 2: Dezoito boards contain individual diamonds (outlined by dashed-lines), with each diamond sub-divided into four triangular cells. The four cells within each diamond must contain numbers between 1 and 8 (may be duplicated) and must sum to 18. With the Solution Guide setting activated, the diamond will turn green when the cells sum to 18. As with Rule 1, this only guarantees that Rule 2 is satisfied.

 

The puzzle challenge begins with many cells pre-defined that can be used as clues to filling in the remaining empty cells. The puzzle challenge is played and won by filling in all remaining cells with numbers between 1 and 8 in accordance with Rules 1 and 2.
